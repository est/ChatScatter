## Context

需要选一个 serverless 运行平台。候选：

- Cloudflare Workers（V8 isolate）
- AWS Lambda
- Vercel Functions
- Deno Deploy

项目是 AI 聊天系统，需要：

- 低延迟（流式响应）
- 全球边缘部署
- 免费额度够开发阶段用
- 不想管服务器

## Decision

选择 Cloudflare Workers。

## Consequences

优点：

- 免费计划每天 10 万次请求，开发阶段绑够用
- 全球 300+ 节点，延迟低
- wrangler CLI 本地开发体验好
- D1 / KV 等存储可按需接入，不用另起服务

缺点：

- V8 isolate 模型有执行时间限制（free 计划 10ms CPU time）
- 不支持 Node.js 原生 API，部分 npm 包不可用
- 冷启动虽快但有上限，超长流式响应需要注意
